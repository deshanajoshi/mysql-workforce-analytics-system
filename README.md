# Corporate Resource & Workforce Analytics System (MySQL)

This project focuses on managing and analyzing workforce data including
employees, skills, projects, resource allocation, and performance analytics.
It is designed for corporate HR and analytics teams.

## Technologies
- MySQL 8.0
- SQL (Advanced)

## Concepts Used
- Relational Database Design
- Joins & Subqueries
- Aggregations
- CASE Statements
- Views
- Business Analytics
